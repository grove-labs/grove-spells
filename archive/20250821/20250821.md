## List of intended changes

### Mainnet
- Controller upgrade
  - Introduce Centrifuge crosschain transfer integration
- Migrate Centrifuge JAAA from V2 to V3
  - Set deposit rate limit max to 0 for Centrifuge JAAA V2
  - Set deposit rate limit slope to 0 for Centrifuge JAAA V2
  - Set redeem rate limit max to 0 for Centrifuge JAAA V2
  - Set redeem rate limit slope to 0 for Centrifuge JAAA V2
  - Set deposit rate limit max to 100M for Centrifuge JAAA V3
  - Set deposit rate limit slope to 50M for Centrifuge JAAA V3
  - Set redeem rate limit max to max uint for Centrifuge JAAA V3
  - Set redeem rate limit slope to 0 for Centrifuge JAAA V3 (not needed with maxed out max)
  - Set crosschain transfer rate limit max to 50M for Centrifuge JAAA V3
  - Set crosschain transfer rate limit slope to 50M for Centrifuge JAAA V3
- Migrate Centrifuge JTRSY from V2 to V3
  - Set deposit rate limit max to 0 for Centrifuge JTRSY V2
  - Set deposit rate limit slope to 0 for Centrifuge JTRSY V2
  - Set redeem rate limit max to 0 for Centrifuge JTRSY V2
  - Set redeem rate limit slope to 0 for Centrifuge JTRSY V2
  - Set deposit rate limit max to 50M for Centrifuge JTRSY V3
  - Set deposit rate limit slope to 50M for Centrifuge JTRSY V3
  - Set redeem rate limit max to max uint for Centrifuge JTRSY V3
  - Set redeem rate limit slope to 0 for Centrifuge JTRSY V3 (not needed with maxed out max)
  - Set crosschain transfer rate limit max to 50M for Centrifuge JTRSY V3
  - Set crosschain transfer rate limit slope to 50M for Centrifuge JTRSY V3
### Avalanche
- Controller upgrade
  - Introduce Centrifuge integration
- Onboard Centrifuge JAAA V3
  - Set deposit rate limit max to 50M for Centrifuge JAAA V3
  - Set deposit rate limit slope to 50M for Centrifuge JAAA V3
  - Set redeem rate limit max to max uint for Centrifuge JAAA V3
  - Set redeem rate limit slope to 0 for Centrifuge JAAA V3 (not needed with maxed out max)
  - Set crosschain transfer rate limit max to 50M for Centrifuge JAAA V3
  - Set crosschain transfer rate limit slope to 50M for Centrifuge JAAA V3
- Onboard Centrifuge JTRSY V3
  - Set deposit rate limit max to 50M for Centrifuge JTRSY V3
  - Set deposit rate limit slope to 50M for Centrifuge JTRSY V3
  - Set redeem rate limit max to max uint for Centrifuge JTRSY V3
  - Set redeem rate limit slope to 0 for Centrifuge JTRSY V3 (not needed with maxed out max)
  - Set crosschain transfer rate limit max to 50M for Centrifuge JTRSY V3
  - Set crosschain transfer rate limit slope to 50M for Centrifuge JTRSY V3
