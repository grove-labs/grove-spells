# 2026-01-15 Grove Spell

## Forum Post

[Forum Post Link](https://forum.sky.money/t/january-15th-2025-proposed-changes-to-grove-for-upcoming-spell/27570)

## List of intended changes

### Ethereum Mainnet

* **Upgrade MainnetController to v1.8.0**
  + Deploys new MainnetController
  + Migrates existing rate limits and configurations
  + Sets backwards compatibility for previously onboarded protocols:
    - sUSDE: maxExchangeRate (1e18, 1.3e18)
    - Curve RLUSD/USDC: maxSlippage 0.9990e18
    - Aave Core RLUSD: maxSlippage 0.9990e18
    - Aave Core USDC: maxSlippage 0.9990e18
    - Aave Horizon RLUSD: maxSlippage 0.9990e18
    - Aave Horizon USDC: maxSlippage 0.9990e18
    - Grove x Steakhouse USDC Morpho Vault: maxExchangeRate (1e18, 1.15e6)

* **Onboard CCTP transfers to Base**
  + domain max 50M (`CCTP_RATE_LIMIT_MAX`)
  + domain slope 50M/day (`CCTP_RATE_LIMIT_SLOPE`)

### Base

* **Initialize Grove Liquidity Layer**
  + Deploy and configure ALM infrastructure (Executor, Receiver, Controller, Proxy, RateLimits, Relayers, Freezer)
  + Configure CCTP mint recipients for Ethereum
  + Configure LayerZero recipients for Ethereum
  + Configure Centrifuge recipients for Ethereum

* **Onboard Morpho Grove x Steakhouse High Yield Vault USDC**
  + deposit max 20M (`GROVE_X_STEAKHOUSE_USDC_MORPHO_VAULT_DEPOSIT_MAX`)
  + deposit slope 20M/day (`GROVE_X_STEAKHOUSE_USDC_MORPHO_VAULT_DEPOSIT_SLOPE`)
  + shareUnit 1e18
  + maxAssetsPerShare 1.15e6
  + withdraw max unlimited
  + withdraw slope 0

* **Onboard CCTP transfers to Ethereum**
  + domain max 50M (`CCTP_RATE_LIMIT_MAX`)
  + domain slope 50M/day (`CCTP_RATE_LIMIT_SLOPE`)
